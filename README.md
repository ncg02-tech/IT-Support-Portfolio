# IT-Support-Portfolio
Hands-on IT support lab showcasing Active Directory setup, domain join troubleshooting, and client/server ticket simulations using Spiceworks. Demonstrates real-world troubleshooting, root cause analysis, and enterprise support workflows.

---

## Projects

### Active Directory Home Lab
Deployed and configured a virtualized Active Directory environment using Windows Server and Windows 11. Implemented domain services, DNS configuration, organizational units, users, and groups to simulate enterprise identity management.
[View Project](./active-directory-home-lab)

---

### Domain Join Troubleshooting
Diagnosed and resolved domain join failures caused by DNS misconfiguration and network communication issues. Utilized command-line tools such as `ping` and `nslookup` to validate connectivity, identify name resolution failures, and restore domain functionality.
[View Project](./domain-join-troubleshooting)

---

### IT Support Ticket Simulation (Client-Side)
Resolved multiple end-user issues on domain-joined Windows 11 systems, including network connectivity failures, shared drive access issues, login problems, and system usability issues (taskbar, desktop, performance). Applied structured troubleshooting using system settings and command-line tools to identify and resolve root causes in a simulated help desk environment.
[View Project](./it-support-ticket-simulation-client-side)

---

### IT Support Ticket Simulation (Server-Side)
Handled administrative-level support tickets on Windows Server, including account lockouts, Active Directory authentication issues, DNS resolution failures, user permission errors, group membership issues, and account management tasks. Performed user provisioning, access control configuration, and directory service troubleshooting using administrative tools and command-line diagnostics.
[View Project](./it-support-ticket-simulation-server-side)
