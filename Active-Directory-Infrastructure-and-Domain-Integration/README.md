# Active Directory Infrastructure and Domain Integration

## Overview
Designed and deployed a virtualized enterprise Active Directory environment using Oracle VirtualBox. Configured a Windows Server 2025 Domain Controller and integrated a Windows 11 client workstation into the domain. Implemented Active Directory Domain Services (AD DS), DNS, organizational structure, user provisioning, group-based access control, and endpoint domain authentication.

Demonstrates hands-on experience with enterprise identity infrastructure, Windows Server administration, network configuration, and Active Directory management. :contentReference[oaicite:0]{index=0}

---

## Environment / Tech Stack
- Oracle VirtualBox
- Windows Server 2025
- Windows 11
- Active Directory Domain Services (AD DS)
- DNS
- TCP/IP Networking
- Windows Server Manager
- Active Directory Users and Computers (ADUC)

---

## Lab Architecture
- Deployed Windows Server 2025 as Domain Controller
- Deployed Windows 11 enterprise client workstation
- Configured isolated VM networking for client/server communication
- Assigned static IP addressing to Domain Controller
- Configured client DNS resolution through Domain Controller
- Implemented centralized domain authentication

---

## Server Configuration
- Installed Active Directory Domain Services (AD DS)
- Promoted Windows Server to Domain Controller
- Configured domain environment and DNS services
- Enabled remote management functionality
- Validated server roles and services

---

## Active Directory Administration
- Created Organizational Units (OUs) to simulate enterprise structure
- Provisioned domain user accounts
- Configured security groups for access management
- Assigned delegated administrative permissions using group membership
- Implemented group-based access control principles

---

## Client Configuration
- Installed and configured Windows 11 client workstation
- Configured network adapter and DNS settings
- Verified connectivity to Domain Controller
- Validated DNS name resolution and network communication

---

## Domain Integration
- Joined Windows 11 workstation to Active Directory domain
- Authenticated using domain credentials
- Verified successful domain login functionality
- Confirmed endpoint integration with domain infrastructure

---

## Key Skills Demonstrated
- Active Directory Administration
- Domain Controller Deployment
- DNS Configuration and Validation
- Windows Server Administration
- Endpoint Domain Integration
- User and Group Management
- Group-Based Access Control (RBAC)
- TCP/IP Network Configuration
- Virtualization (Oracle VirtualBox)
- Enterprise Infrastructure Deployment

---

## Key Takeaways
- DNS configuration is critical for Active Directory functionality
- Proper organizational structure simplifies identity management
- Group-based permissions improve access control scalability
- Endpoint integration depends on proper network and DNS communication
- Structured validation is essential during enterprise infrastructure deployment
